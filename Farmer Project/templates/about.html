About This Project

Hello, I am Pragati Sharma. This project is aimed at helping farmers make informed decisions about which crops to plant based on various input parameters like soil nutrients (N, P, K), temperature, humidity, pH, and rainfall.

Project Development

This project was developed using the following technologies and tools:

  • Python and Flask: For building the web application backend.
  • Bootstrap: For designing a responsive and visually appealing user interface.
  • Machine Learning: A K-Means clustering model was used to predict suitable crops based on the input data. Data preprocessing was done using StandardScaler.
  • HTML/CSS: For structuring and styling the web pages.

How It Works

The user inputs various parameters like soil nutrients, temperature, humidity, pH level, and rainfall. The input data is processed and fed into a pre-trained machine learning model, which predicts the most suitable crops for the given conditions. The results are then displayed on the results page with images and information about each recommended crop.

Benefits for Users

This tool is particularly helpful for farmers who want to optimize their crop yield by selecting the most suitable crops for their land based on scientific data. It takes into account multiple factors that affect crop growth and provides tailored recommendations, making the decision-making process easier and more reliable.

By using this tool, farmers can:

  • Maximize crop yield by selecting the most suitable crops.
  • Make informed decisions based on accurate data analysis.
  • Save time and resources by avoiding trial and error in crop selection.

I hope this project will be a valuable resource for farmers and contribute to more efficient and productive farming practices.
